Channels

  • Channel 1: Color Hue
    • Color represents the attribute of what politcal party women would have voted for
    • Color is an appropriate mapping in thi example because color allows the user to quickly identify which party would have won each state just using the colors. Colors are easy to spot and one of the most effective visualization tools.
  • Channel 2: Shape
    • Shape of graphic allows user to easily see that this is the US
    • This is a good mapping because it shows the viewer each state outlined with a color on which party won in that states electorate

Colormap(s)

  • Colormap 1: Politcal Party (Red and Blue)
    • This red and blue color map shows the attribute of politcal party. Blue for democratic and red for republican and which pary won each state.
    • This is a good mapping in this example because color is a very effective way of showing categorical data like political party. Since there are only two cateogries in this example (Democrat and Republican) it is easy for the viewer to quickly distinguish the two using color.

Channels

  • Channel 1: Color Luminance and Hue
    • Represents the scoring value of each at bat data point and how close each one is to the average.
    • Color mappying is a good mapping choice because it allows the user to quickly identify which data points are the outliers (i.e. home runs) in the visualization.
  • Channel 2: Vertical Position
    • Represents the attribute of the launch angle of each batted ball
    • Vertical positioning here is a good mapping choice because it allows the user to easily view how each data points compares to the other in terms of their outcome as the batting launch angle changes.
  • Channel 3: Horizontal Position
    • Represents the attribute of the exxit velocity of each batted ball
    • Horizontal positioning is a good mapping tool here because it allows the user to quickly identify how the outcome of each at bat changes as exit velocity changes.

Colormap(s)

  • Colormap 1: Average Scoring Value (Purple, Orange, and White)
    • This color map details if batted balls are above or below the average scoring value. If above the data point will be orange and if below the data point will be purple and the saturation changes as the the data point is furhter away from the average.
    • This is a good color mapping becuause it uses two opposite colors to display the differences in the data which helps the viewer quickly identify which data points were the extreme and which area of data points had the most or the least home runs.
